## Examples

### Variant

Use the `variant` prop to control the visual color variation.

```jsx live=true
<Flex direction="column" gap="3">
  <Flex gap="2">
    <Button variant="surface">Button</Button>
    <Button variant="solid">Button</Button>
    <Button variant="subtle">Button</Button>
    <Button variant="outline">Button</Button>
    <Button variant="ghost">Button</Button>
  </Flex>

  <Flex gap="2">
    <Button variant="surface-mono">Button</Button>
    <Button variant="solid-mono">Button</Button>
    <Button variant="subtle-mono">Button</Button>
    <Button variant="outline-mono">Button</Button>
    <Button variant="ghost-mono">Button</Button>
  </Flex>
</Flex>
```

### Size

Use the `size` prop to control the size.

```jsx live=true
<Flex gap="3" align="center">
  <Button size="1">Button</Button>
  <Button size="2">Button</Button>
  <Button size="3">Button</Button>
</Flex>
```
