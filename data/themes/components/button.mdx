## API Reference

This component is based on the `button` element and supports common margin props `m`, `mx`, `my`, `ml`, `mr`, `mt`, `mb`.

<ThemesPropsTable componentName="button" />

## Examples

### Size

Use the `size` prop to control the size.

```jsx live=true
<Flex gap="3" align="center">
  <Button size="1">Button</Button>
  <Button size="2">Button</Button>
  <Button size="3">Button</Button>
</Flex>
```

### Variant

Use the `variant` prop to control the visual style.

```jsx live=true
<Flex direction="column" gap="3">
  <Flex gap="3">
    <Button variant="surface">Surface</Button>
    <Button variant="solid">Solid</Button>
    <Button variant="subtle">Subtle</Button>
    <Button variant="outline">Outline</Button>
    <Button variant="ghost">Ghost</Button>
  </Flex>

  <Flex gap="3">
    <Button variant="surface-mono">Surface</Button>
    <Button variant="solid-mono">Solid</Button>
    <Button variant="subtle-mono">Subtle</Button>
    <Button variant="outline-mono">Outline</Button>
    <Button variant="ghost-mono">Ghost</Button>
  </Flex>
</Flex>
```

### Color

Use the `color` prop to assign a specific color value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <Button color="crimson">Crimson</Button>
  <Button color="indigo">Indigo</Button>
  <Button color="grass">Grass</Button>
  <Button color="orange">Orange</Button>
</Flex>
```

### Radius

Use the `radius` prop to assign a specific radius value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <Button radius="none">None</Button>
  <Button radius="small">Small</Button>
  <Button radius="medium">Medium</Button>
  <Button radius="large">Large</Button>
  <Button radius="full">Full</Button>
</Flex>
```
