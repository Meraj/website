```jsx live=true
<TextField placeholder="The quick brown fox jumped" />
```

## API Reference

This component is based on the `input` element and supports common margin props `m`, `mx`, `my`, `ml`, `mr`, `mt`, `mb`.

<ThemesPropsTable componentName="textfield" />

## Examples

### Size

Use the `size` prop to control the size.

```jsx live=true
<Grid gap="3">
  <TextField size="1" placeholder="The quick brown fox jumped" />
  <TextField size="2" placeholder="The quick brown fox jumped" />
  <TextField size="3" placeholder="The quick brown fox jumped" />
</Grid>
```

### Variant

Use the `variant` prop to control the visual style.

```jsx live=true
<Grid columns="3" gap="3">
  <TextField variant="surface" placeholder="The quick brown fox jumped" />
  <TextField variant="surface-mono" placeholder="The quick brown fox jumped" />
  <TextField variant="subtle-mono" placeholder="The quick brown fox jumped" />
</Grid>
```

### Radius

Use the `radius` prop to assign a specific radius value, ignoring the global theme.

```jsx live=true
<Grid columns="5" gap="3">
  <TextField radius="none" placeholder="The quick brown fox jumped" />
  <TextField radius="small" placeholder="The quick brown fox jumped" />
  <TextField radius="medium" placeholder="The quick brown fox jumped" />
  <TextField radius="large" placeholder="The quick brown fox jumped" />
  <TextField radius="full" placeholder="The quick brown fox jumped" />
</Grid>
```
