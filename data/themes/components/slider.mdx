---
metaTitle: Slider
metaDescription: Provides user selection from a range of values.
---

```jsx live=true
<Slider defaultValue={[50]} />
```

## API Reference

This component inherits the props from the [Radix Slider](/docs/primitives/components/slider) primitive and supports common margin props `m`, `mx`, `my`, `ml`, `mr`, `mt`, `mb`.

<ThemesPropsTable name="slider" />

## Examples

### Size

Use the `size` prop to control the size.

```jsx live=true
<Flex direction="column" gap="4">
  <Slider size="1" defaultValue={[25]} />
  <Slider size="2" defaultValue={[50]} />
  <Slider size="3" defaultValue={[75]} />
</Flex>
```

### Variant

Use the `variant` prop to control the visual style.

```jsx live=true
<Grid columns="1" gap="4">
  <Slider variant="solid" defaultValue={[50]} />
  <Slider variant="solid-mono" defaultValue={[50]} />
  <Slider variant="subtle" defaultValue={[50]} />
  <Slider variant="subtle-mono" defaultValue={[50]} />
</Grid>
```

### Color

Use the `color` prop to assign a specific color value, ignoring the global theme.

```jsx live=true
<Grid columns="1" gap="4">
  <Slider color="crimson" defaultValue={[50]} />
  <Slider color="indigo" defaultValue={[50]} />
  <Slider color="grass" defaultValue={[50]} />
  <Slider color="orange" defaultValue={[50]} />
</Grid>
```

### Radius

Use the `radius` prop to assign a specific radius value, ignoring the global theme.

```jsx live=true
<Grid columns="5" gap="4">
  <Slider radius="none" defaultValue={[75]} />
  <Slider radius="small" defaultValue={[75]} />
  <Slider radius="medium" defaultValue={[75]} />
  <Slider radius="large" defaultValue={[75]} />
  <Slider radius="full" defaultValue={[75]} />
</Grid>
```
