---
metaTitle: Icon Button
metaDescription: Use to present an icon within a button.
---

```jsx live=true
<IconButton>
  <MagnifyingGlassIcon />
</IconButton>
```

## API Reference

This component is based on the `button` element and supports common margin props `m`, `mx`, `my`, `ml`, `mr`, `mt`, `mb`.

## Examples

### Size

Use the `size` prop to control the size.

```jsx live=true
<Flex gap="3" align="center">
  <IconButton size="1">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton size="2">
    <MagnifyingGlassIcon width="18" height="18" />
  </IconButton>
  <IconButton size="3">
    <MagnifyingGlassIcon width="22" height="22" />
  </IconButton>
</Flex>
```

### Variant

Use the `variant` prop to control the visual style.

```jsx live=true
<Flex direction="column" gap="3">
  <Flex gap="3">
    <IconButton variant="surface">
      <MagnifyingGlassIcon />
    </IconButton>
    <IconButton variant="solid">
      <MagnifyingGlassIcon />
    </IconButton>
    <IconButton variant="subtle">
      <MagnifyingGlassIcon />
    </IconButton>
    <IconButton variant="outline">
      <MagnifyingGlassIcon />
    </IconButton>
    <IconButton variant="ghost">
      <MagnifyingGlassIcon />
    </IconButton>
  </Flex>

  <Flex gap="3">
    <IconButton variant="surface-mono">
      <MagnifyingGlassIcon />
    </IconButton>
    <IconButton variant="solid-mono">
      <MagnifyingGlassIcon />
    </IconButton>
    <IconButton variant="subtle-mono">
      <MagnifyingGlassIcon />
    </IconButton>
    <IconButton variant="outline-mono">
      <MagnifyingGlassIcon />
    </IconButton>
    <IconButton variant="ghost-mono">
      <MagnifyingGlassIcon />
    </IconButton>
  </Flex>
</Flex>
```

### Color

Use the `color` prop to assign a specific color value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <IconButton color="crimson">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton color="indigo">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton color="grass">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton color="orange">
    <MagnifyingGlassIcon />
  </IconButton>
</Flex>
```

### Radius

Use the `radius` prop to assign a specific radius value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <IconButton radius="none">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton radius="small">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton radius="medium">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton radius="large">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton radius="full">
    <MagnifyingGlassIcon />
  </IconButton>
</Flex>
```
