## API Reference

This component inherits the props from the [Radix Switch](/docs/primitives/components/switch) primitive and supports common margin props `m`, `mx`, `my`, `ml`, `mr`, `mt`, `mb`.

## Examples

### Size

Use the `size` prop to control the size.

```jsx live=true
<Flex gap="3">
  <Switch size="1" />
  <Switch size="2" />
  <Switch size="3" />
</Flex>
```

### Variant

Use the `variant` prop to control the visual style.

```jsx live=true
<Flex gap="3">
  <Switch variant="solid" defaultChecked />
  <Switch variant="solid-mono" defaultChecked />
</Flex>
```

### Color

Use the `color` prop to assign a specific color value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <Switch color="crimson" defaultChecked />
  <Switch color="indigo" defaultChecked />
  <Switch color="grass" defaultChecked />
  <Switch color="orange" defaultChecked />
</Flex>
```

### Radius

Use the `radius` prop to assign a specific radius value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <Switch radius="none" />
  <Switch radius="small" />
  <Switch radius="medium" />
  <Switch radius="large" />
  <Switch radius="full" />
</Flex>
```
