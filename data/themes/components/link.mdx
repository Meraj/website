## API Reference

This component is based on the `a` element and supports common margin props `m`, `mx`, `my`, `ml`, `mr`, `mt`, `mb`.

### Variant

Use the `variant` prop to control the visual style.

```jsx live=true
<Flex direction="column" gap="1">
  <Link variant="default">The quick brown fox</Link>
  <Link variant="high-contrast">The quick brown fox</Link>
</Flex>
```

### Weight

Use the `weight` prop to set the weight of the link. You can change the weight across breakpoints using the object syntax.

```jsx live="true"
<Flex direction="column" gap="1">
  <Link weight="normal">The quick brown fox</Link>
  <Link weight="bold">The quick brown fox</Link>
</Flex>
```

### Color

Use the `color` prop to assign a specific color value, ignoring the global theme.

```jsx live=true
<Flex direction="column" gap="1">
  <Link color="crimson">The quick brown fox</Link>
  <Link color="indigo">The quick brown fox</Link>
  <Link color="grass">The quick brown fox</Link>
  <Link color="orange">The quick brown fox</Link>
</Flex>
```
