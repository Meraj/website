---
metaTitle: Sup
metaDescription: Marks text to display as superscript.
---

```jsx live=true
<Text>
  a<Sup>2</Sup> + <Sup>b2</Sup> = <Sup>c2</Sup>
</Text>
```

## API Reference

This component is based on the `sup` element and supports common margin props `m`, `mx`, `my`, `ml`, `mr`, `mt`, `mb`.
