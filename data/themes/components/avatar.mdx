### Variant

Use the `variant` prop to control the visual style.

```jsx live=true
<Flex gap="3">
  <Avatar variant="solid" />
  <Avatar variant="solid-mono" />
  <Avatar variant="subtle" />
  <Avatar variant="subtle-mono" />
</Flex>
```

### Color

Use the `color` prop to assign a specific color value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <Avatar color="crimson" />
  <Avatar color="indigo" />
  <Avatar color="grass" />
  <Avatar color="orange" />
</Flex>
```

### Radius

Use the `radius` prop to assign a specific radius value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <Avatar radius="none" />
  <Avatar radius="small" />
  <Avatar radius="medium" />
  <Avatar radius="large" />
  <Avatar radius="full" />
</Flex>
```

```jsx live=true
          0112 22.5c-2.786 0-5.433-.608-7.812-1.7a.75.75 0 01-.437-.695z"
        />
      </svg>
    }
  />
</Flex>
```
