## API Reference

This component inherits the props from the [Radix Radio Group](/docs/primitives/components/radio-group) primitive and supports common margin props `m`, `mx`, `my`, `ml`, `mr`, `mt`, `mb`.

### Item

An item in the group that can be checked. An input will also render when used within a form to ensure events propagate correctly.

## Examples

### Variant

Use the `variant` prop to control the visual color variation.

```jsx live=true
<Flex gap="6">
  <RadioGroup.Root variant="solid" defaultValue="1">
    <Flex gap="1" direction="column">
      <label>
        <Flex gap="2" align="center">
          <RadioGroup.Item value="1" />
          <Text>Default</Text>
        </Flex>
      </label>
    </Flex>
  </RadioGroup.Root>

  <RadioGroup.Root variant="solid-mono" defaultValue="1">
    <Flex gap="1" direction="column">
      <label>
        <Flex gap="2" align="center">
          <RadioGroup.Item value="1" />
          <Text>Default</Text>
        </Flex>
      </label>
    </Flex>
  </RadioGroup.Root>
</Flex>
```
