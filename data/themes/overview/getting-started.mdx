## Setting up

### 1. Import the CSS file

<Note>
  Note: This may vary depending of your framework of choice, but generally
  you'll want to import the CSS file at the root of your application.
</Note>

```ts
import '@radix-ui/themes/dist/index.css';
```

### 2. Wrap your app with the ThemeConfig

```tsx
import { ThemeConfig } from '@radix-ui/themes';

export default function () {
  return (
    <ThemeConfig>
      <YourApp />
    </ThemeConfig>
  );
}
```

## Usage

You are now ready to use Radix's components in your app:

```tsx
import { Flex, Text, Button } from '@radix-ui/themes';

export default function MyApp() {
  return (
    <Flex direction="column" gap="2">
      <Text>Hello from Radix Themes :)</Text>
      <Button>Get started</Button>
    </Flex>
  );
}
```
